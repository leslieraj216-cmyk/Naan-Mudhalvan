 Product Catalog with Filters (HTML, CSS, JS)

A simple and responsive **Product Catalog** built using **pure HTML, CSS, and JavaScript**.  
Users can browse, search, and filter products by category, price range, and rating — all done on the client side with no backend required.

---

 Features

-  **Product Grid** – Displays products with image, title, price, and rating  
-  **Search Functionality** – Real-time search by product name or keyword  
-  **Filter Options** – Filter by category, price range, and rating  
-  **Sorting** – Sort products by price (low-to-high, high-to-low)  
- **Responsive Layout** – Works across all screen sizes  
-  **Frontend Only** – No backend or API, just static data  



 Tech Stack

- **HTML5** – Structure  
- **CSS3** – Styling (Flexbox / Grid)  
- **JavaScript (ES6)** – Filtering, search, and rendering logic
